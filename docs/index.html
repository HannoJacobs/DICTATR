Getting started

  1. Open the DMG and drag DICTATR to Applications
  2. Open DICTATR from Applications. If Gatekeeper blocks the first launch, right-click the app and choose Open.
  3. Grant Microphone access when prompted
  4. Open System Settings, search "Privacy", select Privacy & Security → Accessibility, and toggle DICTATR on
  5. After reinstalling or updating an ad-hoc build, expect to re-enable Accessibility and possibly re-grant Microphone access
  6. Press fn+F5 anywhere and start talking
